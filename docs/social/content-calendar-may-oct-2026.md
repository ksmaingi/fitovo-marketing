# Fitovo 6-Month Post-Launch Content Calendar

**Covers:** May–October 2026 (launch month through Month 6)
**Status:** Strategic editorial plan. The weekly Claude scheduled task generates actual scripts each Monday based on this skeleton + performance data from the prior week. Don't treat this as a script — treat it as the season arc.

---

## ⚠️ READ FIRST — rescaled to 3-hour-per-week time budget

This calendar was originally scoped at 5 Reels + 2 Carousels + 21 Stories/week. **That is not achievable at Kulvinder's 3-hr/week time budget.** After the Week 3 diagnostic (Apr 2026), cadence is rescaled to:

- **2 Reels/week** — Tuesday + Friday, 7:30 AM
- **1 Carousel/week** — Saturday 9:00 AM (Gamma-generated)
- **5 Stories/week** — one per weekday, 2-min phone + text overlay
- **No weekend posting** (except Sat Carousel) unless opportunistic

### How to read this calendar at reduced cadence

Each week below lists 5 topic ideas across pillars. **The Claude scheduled task picks 2 to develop into full Reels each week** (rotating pillars so no single pillar drops), plus 1 Carousel topic. The remaining topics become:
- Future weeks' rotation (evergreen — not time-sensitive)
- Story content (distilled to a single text-overlay insight)
- Skipped without guilt

**The goal is 2 excellent Reels per week, not 5 rushed ones.** Volume above actual capacity ships nothing; low volume executed well ships consistently.

## How the weekly Claude scheduled task uses this calendar

1. Every Monday at 7 AM (configured in `marketing-plan.md` §6), the scheduled task reads:
   - This file (for the current week's topic row — treats 5 listed topics as candidates)
   - `docs/marketing-plan.md` §6 prompt + brand voice
   - Last week's top/bottom Reel performance (if available) to avoid repeating hooks
   - The "previous hooks to avoid" list from the last 2 weeks of generated calendars
2. The agent **picks 2 of the 5 candidate topics** (rotating pillars so no single pillar drops for 2 weeks in a row) and generates:
   - 2 full Reels (hook + script + caption + hashtags + visual notes)
   - 1 Carousel outline (for Gamma input)
   - 5 Story concepts (one per weekday)
3. Kulvinder reviews Monday evening in ~30 min. Tweaks tone, factual checks on studies, confirms CTA.
4. Output merged with `scripts/` content-calendar scripts for summary.md.

**This calendar gives the agent the lane, not the script.** The agent writes hooks, scripts, captions — grounded by: the month's theme, the week's focus, the pillars to rotate through, and the avoid-list.

## Review cadence

- **Weekly:** Is this week's topic still right given last week's data? (Monday, 5 min, in `weekly-review.md`.)
- **Monthly:** Did the month's theme land? Was there a better theme the audience revealed? (First Monday, 15 min.)
- **Quarterly:** Full rewrite of the remaining months in this file. (First week of quarter, 2 hours.)

This file is meant to be edited. If a month's theme isn't working by Week 2 of it, change the theme for Weeks 3–4 — don't ride a plan that's drifted from reality.
